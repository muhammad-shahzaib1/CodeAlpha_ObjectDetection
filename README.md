# Object Detection System

This project was developed as part of the CodeAlpha Artificial Intelligence Internship.

## Project Description:
The Object Detection System is a Python-based computer vision application that performs real-time face detection using OpenCV and Haar Cascade classifiers.

The system captures webcam video and detects faces automatically by drawing bounding boxes around them.

---

## Features:
- Real-time face detection
- Webcam video processing
- Bounding box visualization
- Beginner-friendly computer vision project
- Lightweight and fast detection

---

## Technologies Used:
- Python
- OpenCV
- Haar Cascade Classifier

---

## How to Run the Project

Install required libraries:
```bash
pip install opencv-python
```

Run the application:
```bash
python object_detection.py
```
